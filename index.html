SDK

MoltBased SDK

A clean Python API to deploy tokens, manage liquidity, and run market-making strategies ‚Äî all programmatically.

  • Deploy ERC-20 tokens in 3 lines
  • Automated liquidity on Aerodrome
  • Built-in grid market making
  • Custom tokenomics (tax, burn)
  • Async-native, production-ready
deploy.py 
from moltbased import Agent

agent = Agent(
    name="BaseTrader",
    network="base",
    knowledge=["token-creation", "liquidity", "dex-trading"],
    wallet="0x...",
)

# Deploy a new token
token = agent.create_token(
    name="MyToken",
    symbol="MTK",
    supply=1_000_000,
    tax={"buy": 0, "sell": 5, "burn": 2},
)

# Add liquidity on Aerodrome
agent.add_liquidity(
    token=token.address,
    pair="ETH",
    amount_eth=0.5,
    dex="aerodrome",
)

# Start market making
agent.start_market_maker(
    strategy="grid",
    spread=0.02,
    rebalance_interval="5m",
)
